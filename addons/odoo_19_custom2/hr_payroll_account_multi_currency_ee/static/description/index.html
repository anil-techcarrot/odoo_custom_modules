Payroll Accounting Multi Currency

Note: This module works only in Odoo Enterprise

Overview

This module allow you to handle multiple currencies in payroll accounting.

Features
Selecting currency when creating new contract.
The payslip currency will be automatically set based on contract currency.
The attachment amount will be converted to the payslip currency.
The amount will appear in journal entry with the "amount currency".
How it works

Payroll Accounting Multi Currency

Selecting currency when creating new contract:
When you create a new contract you will see currency field in the form , and when you change the currency, the currency symbol will be changed as shown below:

The payslip currency will be automatically set based on contract currency:
When you create a payslip the payslip currency will be changed automatically based on the selected contract currency, in the image below we choose the contract with dollar $ currency for the payslip.
If the employee has attachment, they will be converted from the company currency to the payslip currency.

When you click on compute sheet button the payslip computation will be computed with the selected contract currency.

When you create a draft entry for the payslip you can see the amount with company currency under the debit and credit as usual , in addition also to the amount currency and currency of the payslip as shown below.

If you want to refund the payslip the refunded payslip calculation will be reversed as normal based on the original payslip currency.
